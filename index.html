Bolche-Bot,

el bot de discord oficial de la Unión Soviética Espartana.

Descripción:

El Bolche-Bot es un bot de discord programado en JavaScript, a través de la librería "Discord.js"

Historia:

La idea de programar un bot surgió, como muchas cosas, por necesidad. El plan original era tener un bot que detectáse cuando alguien estuviése en un canal de voz, y usaría un comando de otro bot para poner una canción al azar. Lamentablemente esto no se pudo lograr, no tenía (ni tengo) los conocimientos necesarios para que el bot detectáse que alguien esté en el canal de voz, y además, los bots no toman en cuenta los mensajes de otros bots, por lo que tampoco podría usar comandos. Me di por vencido con esa idea, pero de todos modos continué con el desarrollo del bot, agregando nuevas funciones y chistes, por que ese es el objetivo principal, el entretenimiento.

Comandos:

Utilidad

  • !should => Si deberías o no jugar a algo, con 50% de probabilidades. Sirve cualquier argumento, solo se usa la primera palabra.
    Ej: !should Skyrim, responde Deberías jugar al Skyrim.

  • !decide => Decide entre los dos argumentos entregados.
    Ej: !decide ekko lee, responde ekko

  • !ranNum => Devuelve un número al azar que se encuentre entre los valores especificados. Solo se pueden ocupar números como argumentos.
    Ej: !ranNum 7 14, responde 11

  • !avatar => Responde con tu avatar de Discord.
    Ej: !avatar, responde Ten, la siguiente vez serán 2 dólares. [foto del avatar]

  • !build => Devuelve el link de League of Graphs del campeón especificado. El argumento tiene que ser el nombre del campeón en inglés.
    Ej: !build ornn, responde Aquí está el resultado de tu busqueda: [link]

  • !badlofi => Música compuesta por el admin.

Entretenimiento:

  • !game => Probabilidades de ganar el game. Argumentos: carry, noCarry y tryndaNoIgnite.
    Ej: !game noCarry, responde 55% de probabilidades de ganar el game.

  • !puente => Probabilidades de tirarse de un puente: Argumentos: vicho (también funciona sin argumento).
    Ej: !puente, responde Las probabilidades de que @nombre se tire de un puente son de 31 %

  • !cringe => Indica que lo que acaba de suceder fue cringe.
    Ej: !cringe, responde Definitivamente cringe(en tts)

  • !gg => El bot te felicita a ti o a alguna persona que especifiques en el argumento.
    Ej: !gg bot, responde Omedetou, bot-kun

  • !succ y !succf => (Argumento) la chupa. La variación con "f" sirve para la versión femenina.

  • !music => Se explica solo, no necesita argumentos.

  • !admin => Mensaje en TTS del bot.

  • !respect => jaja imagen chistosa.

  • !cumple => El bot le desea un feliz cumpleaños a la persona especificada.

  • !buy => Comprarle un producto al bot, actualmente solo tiene empanada de pino. Argumentos: empanada.

Otras Funciones:

  • probabilidades => Si comienzas el mensaje con esta palabra, el bot responderá las probabilidades de que eso ocurra.
    Ej: probabilidades de que el comando funcione, responde 33% probabilidades de que el comando funcione.

  • debería => Si comienzas el mensaje con esta palabra, el bot responderá si deberías hacer esa acción.
    Ej: debería seguir escribiendo el código de esta página?
